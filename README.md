# TaboolaManagement
 This project is Taboola & Google Analysitc API Integration.
 
## Publish Url
 https://system.smartpublishers.co/


## Get Started
env setting

Composer install
```sh
$ composer install
```

Laravel license
```sh
$ php artisan key:generate
```

Database migrate and seed
```sh
$ php artisan migrate --seed
```

Storage link
```sh
$ php artisan storage:link
```

Run serve
```sh
$ php artisan serve
```
